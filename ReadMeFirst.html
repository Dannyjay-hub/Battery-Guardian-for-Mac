Install Guide

1
Drag to Applications
In the installer window, drag Battery Guardian into the Applications folder. Then eject the disk image.
2
Open the app for the first time
Go to Applications, then right-click Battery Guardian and select Open from the menu. When the dialog appears, click Open. You only need to do this once.

If macOS says "Battery Guardian cannot be verified" — do not click "Move to Trash." Click Cancel, then right-click the app and select Open instead.

If that does not work: System Settings → Privacy & Security → scroll down and click "Open Anyway".
3
Run your scan
The app scans automatically when it opens. Click Rescan at any time to run it again. No internet connection required.
Your data never leaves your Mac
Battery Guardian reads hardware registers directly using ioreg — a built-in macOS system call. No accounts, no telemetry, no internet connection needed. Everything runs locally.
